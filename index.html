📲 Jak zaimportować plik .ics?

Aplikacja Sposób importu
Google CalendarOtwórz calendar.google.com → Ustawienia (⚙️) → Importuj → wybierz plik .ics
Apple Calendar (Mac)Menu Plik → Importuj → wybierz plik .ics
Apple Calendar (iPhone/iPad)Wyślij plik na e-mail lub AirDrop → otwórz w Kalendarzu
Outlook (desktop)Plik → Otwórz i eksportuj → Importuj/Eksportuj → Importuj plik iCalendar
Outlook (web)Otwórz outlook.live.com → Dodaj kalendarz → Prześlij z pliku → wybierz .ics
ThunderbirdZakładka Kalendarz → Plik → Importuj → wybierz plik .ics
