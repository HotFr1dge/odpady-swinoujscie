## 📲 Jak zaimportować plik .ics?

| Aplikacja | Sposób importu |
|-----------|---------------|
| **Google Calendar** | [calendar.google.com](https://calendar.google.com) → Ustawienia ⚙️ → Importuj → wybierz plik `.ics` |
| **Apple Calendar (Mac)** | Menu Plik → Importuj → wybierz plik `.ics` |
| **Apple Calendar (iPhone/iPad)** | Wyślij plik na e-mail lub przez AirDrop → otwórz w aplikacji Kalendarz |
| **Outlook (desktop)** | Plik → Otwórz i eksportuj → Importuj/Eksportuj → Importuj plik iCalendar |
| **Outlook (web)** | [outlook.live.com](https://outlook.live.com) → Dodaj kalendarz → Prześlij z pliku |
| **Thunderbird** | Zakładka Kalendarz → Plik → Importuj → wybierz plik `.ics` |

Po jednorazowym imporcie wszystkie terminy pojawią się w Twoim kalendarzu — nie musisz nic aktualizować przez cały 2026 rok.
